World Factbook Site

This project contains files for the following:

/site ......... static web site for browsing/searching data, available online at https://worldfactbook.blob.core.windows.net/site/index.html
/functions .... Azure Functions source code and configuration screenshots, entered via Azure Portal

For a full description of this project, see David Pallmann's blog
